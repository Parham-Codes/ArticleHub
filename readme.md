# рҹҡҖ Getting Started

## Clone Repository

```bash
git clone https://github.com/Parham-Codes/ArticleHub.git
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

## Start JSON Server

```bash
npx json-server db.json
```

Application will be available at:

```
http://localhost:5173
```

JSON Server:

```
http://localhost:3000
```
