# ✨ Features

- 📄 View all articles
- 🔍 Read article details
- ➕ Create new articles
- ✏️ Edit existing articles
- 🗑️ Delete articles
- 📱 Fully Responsive Design
- ⚡ Fast performance powered by Vite
- 🌐 REST API communication using Axios
- 🛣️ Client-side routing with React Router
- 💬 Beautiful confirmation dialogs using SweetAlert2
- ⏳ Loading Component
- 🔢 Animated Statistics (PureCounter)
- 🚫 Custom 404 Page
- ⬆️ Scroll To Top functionality
- 🎨 Clean and reusable component architecture

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Parham-Codes/ArticleHub.git
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

## Start JSON Server

```bash
npx json-server db.json
```

Application will be available at:

```
http://localhost:5173
```

JSON Server:

```
http://localhost:3000
```
